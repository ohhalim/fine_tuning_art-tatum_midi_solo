# D0 진단 실험 결과 — 데이터 레짐 vs 다양성 붕괴 (2026-07-11 실행)

이슈 #1444 / PR #1445. 하니스: `scripts/run_d0_experiment.sh` (MPS, 기본 파라미터로 실행).

## 실험 설계

동일 아키텍처(13.4M 파라미터, from-scratch `--train_full_model`), 동일 하이퍼파라미터(8 epoch)에서
**학습 데이터만** 바꾼 두-팔 대조 실험:

- **Arm A**: probe 16조각 (기존 붕괴 재현 조건)
- **Arm B**: 전체 코퍼스 2,777조각 (train 2,499)

각 팔의 체크포인트로 동일 프라이머·온도에서 24샘플씩 생성 후 `diversity_metrics.py`로 비교.

## 판정

**병목은 base 표현력(아키텍처)이 아니라 데이터 레짐이다.** 세 층의 증거가 일치한다.

### 1. 학습 손실

| | Arm A (16조각) | Arm B (2,777조각) |
|---|---|---|
| best val loss | **5.0585** (epoch 6) | **3.1934** (epoch 8) |
| perplexity | ≈157 | ≈24 (**6.6× 개선**) |
| 거동 | epoch 6 바닥 후 반등 (5.06→5.17→5.20, 과적합) | 8 epoch 내내 단조 하강 — 학습 여지 잔존 |

epoch별 val loss:

| epoch | 1 | 2 | 3 | 4 | 5 | 6 | 7 | 8 |
|---|---|---|---|---|---|---|---|---|
| Arm A | 5.768 | 5.609 | 5.333 | 5.256 | 5.133 | **5.059** | 5.169 | 5.199 |
| Arm B | 4.353 | 4.169 | 3.882 | 3.557 | 3.412 | 3.309 | 3.239 | **3.193** |

### 2. 생성 샘플 다양성 (팔당 24샘플)

| 지표 | Arm A | Arm B | 변화 |
|---|---|---|---|
| 고유 보이싱 / 샘플 | 14.3 | 21.4 | +49% |
| 보이싱 엔트로피 (bits) | 3.60 | 4.00 | +0.40 |
| 풀링 고유 보이싱 | 81 | 145 | +79% |
| 노트 밀도 (/초) | 0.79 | 1.77 | ×2.2 |
| 노트 수 / 샘플 | 31.9 | 68.2 | ×2.1 |
| 피치클래스 엔트로피 | 3.29 | 3.30 | 동률 |

### 3. 데이터 상한 대비 (풀링 보이싱 엔트로피)

| | 데이터 상한 | 생성 | 도달률 |
|---|---|---|---|
| Arm A | 4.94 bits | 4.61 bits | **93.3%** |
| Arm B | 6.29 bits | 5.00 bits | 79.5% |

**Arm A는 자기 데이터의 다양성 상한에 사실상 도달해 있었다.** 모델이 부족했던 게 아니라
데이터의 천장이 낮았던 것이다. Arm B는 상한이 1.35 bits 높고 도달률이 79%라 연장 학습 시
추가 개선 여지가 있다.

## 해석 시 주의점

1. **distinct_voicing_ratio(샘플당)는 Arm A가 높다 (0.565 vs 0.431)** — 길이 아티팩트.
   Arm A는 샘플당 보이싱이 절반(25.7 vs 50.8)이라 적게 뽑을수록 고유 비율이 자연히 높다.
   풀링 기준 사실상 동률(0.131 vs 0.119)이며 절대량(81 vs 145)이 올바른 비교다.
2. **mean_voicing_size 1.2–1.3** — 두 팔 모두 생성이 거의 단선율. Tatum 스타일 화성 밀도와는
   아직 거리가 멀다. D0는 진단 실험이므로 판정에는 무관하나 다음 단계의 품질 과제.
3. **단일 시드, 24샘플, 8 epoch** — 효과 크기(ppl 6.6×)가 커서 판정은 견고하지만
   정량 주장에는 시드 반복 필요.
4. **decode 시 "removed pitch" 로그** — 유효 음역(21–108) 안의 값도 제거로 찍힌 사례 있음.
   디코드 경로 확인 필요 (지표는 살아남은 노트 기준이라 판정 무관).

## 다음 방향

1. **2단계 커리큘럼으로 본 목표 복귀**: 전체 코퍼스 사전학습 → Tatum 서브셋 파인튜닝.
   16조각 직접 학습이 붕괴한 원인이 데이터 레짐이므로, 넓은 분포를 먼저 배우고 스타일을
   얹는 구조가 정공법. 전조(transposition) 증강으로 Tatum 유효 데이터 확장 병행.
2. **생성 품질 트랙**: voicing size(단선율) 문제, removed-pitch 디코드 이슈 조사.
3. **(선택) Arm B 연장 학습**: val 하강 중이므로 16 epoch 연장 시 상한 도달률 상승 확인.
