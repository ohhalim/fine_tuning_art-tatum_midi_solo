# D1 실험 결과: LoRA 커리큘럼이 스타일 FT의 다양성 붕괴를 막는다

> Issue #1449 / 설계 `D1_DESIGN.md`(#1448, 판정 기준 사전 등록) / 하니스 `scripts/run_d1_experiment.sh`(#1450, #1452)
> 실행: 2026-07-12, 로컬 MPS. 모든 팔 seed 42, 생성 조건 D0와 동일(24샘플, temp 1.0, len 1024, 동일 프라이머).

## 질문

넓은 코퍼스(2,777) 사전학습 → 스타일 서브셋(Mehldau 16) 파인튜닝이,
from-scratch 스타일 학습(Arm A)에서 잃었던 다양성을 유지하면서 스타일 적응이 되는가?

## 결과 — 사전 등록 기준표에 실측값 대입

주 판정: pooled_unique_voicings (고정 참조 A=81, B=145 / 강한 성공 ≥116, 부분 105–115, 실패 <105)

| 팔 | 구성 | pooled 고유 보이싱 | pooled 엔트로피 (bits) | 노트 밀도 /s | best probe val | **판정** |
|---|---|---|---|---|---|---|
| A | from-scratch, probe 16 | 81 | 4.61 | 0.79 | 5.0585 | (붕괴 대조군) |
| B | 사전학습만, FT 없음 | 145 | 5.00 | 1.77 | — | (참조 상한) |
| C | B → **full FT** (lr 3e-5, 4ep) | 108 | 4.53 | 1.70 | 3.104 | **부분 성공** |
| **D** | B → **LoRA-only** (lr 3e-4, 8ep) | **142** | **5.03** | 1.84 | 3.136 | **강한 성공** |
| E | B → full FT + 전조 증강 ×12 (194 seq) | **80** | 4.21 | 1.27 | 3.208* | **실패** |

\* Arm E의 val은 증강 val(같은 2곡의 12키 전조)이라 다른 팔의 probe val과 완전 동일 셋은 아님 — 게이트 해석에만 참고.

스타일 적응 게이트 (probe val loss < 5.0585 + 학습 중 하강): **C, D 모두 통과**
(C: 3.35→3.10 하강 후 4ep 반등 / D: 3.37→3.14 하강). E도 하강(3.28→3.21)으로 적응 자체는 발생.

## 판정

1. **H-D1 (주 가설) 입증 — 커리큘럼이 붕괴를 막는다. 단, LoRA로 할 때.**
   Arm D는 다양성을 Arm B의 98%(142/145) 수준으로 유지하면서 probe val loss를
   5.06 → 3.14로 낮췄다(Arm A와 같은 val 2파일 기준). 다양성과 스타일 적응이 양립한다.

2. **H-D1a 입증 — LoRA(base 동결) ≫ full FT.**
   적응력은 사실상 동급(best val 3.136 vs 3.104)인데 다양성 보존은 142 vs 108.
   base를 동결하면 사전학습 지식이 구조적으로 보호된다.

3. **H-D1b 반증 — 전조 증강은 full FT에서 역효과.**
   Arm E(194 seq, 총 196 스텝)는 C(16 seq, 총 16 스텝)보다 좁은 스타일 그래디언트를
   12배 더 받아 사전학습 지식을 더 많이 덮어썼고, 다양성이 Arm A 수준(80)까지 붕괴했다.
   full FT에서는 "스타일 데이터가 많을수록 더 붕괴"한다 — 붕괴의 원인이 데이터 양이 아니라
   **사전학습 분포에서의 이탈량**임을 보여주는 대비 증거.

**종합: 스타일 FT의 정답 레시피 = 넓은 코퍼스 사전학습 → LoRA-only 스타일 적응.**
D0("좁은 데이터 직접 학습은 붕괴")과 D1("사전학습+LoRA는 붕괴 없이 적응")이 하나의
일관된 그림을 완성한다.

## 한계 / 주의

- **생성은 최신 epoch 체크포인트 사용** (C: ep4, D: ep8 — best val epoch 아님). D0와 동일한
  하니스 규약이라 팔 간 비교는 공정하나, best-epoch 생성 시 수치가 더 좋아질 여지 있음.
- 단일 시드(42), 24샘플 — D의 142 vs C의 108은 격차가 커서 견고하나, C(108)와
  부분/실패 경계(105)는 근소함.
- Arm B 체크포인트의 LoRA 가중치가 이미 학습된 상태에서 D가 이어서 학습함 (설계서 §6-2).
- removed-pitch 디코드 이슈(#1451)는 전 팔 공통 — 비교 유효, 절대값 해석만 주의.
- Arm E의 val 셋이 증강본이라 게이트 수치의 직접 비교 불가(위 표 각주).
